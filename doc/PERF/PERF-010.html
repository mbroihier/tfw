Test Case PERF-010
Title rtnv meets its time and capacity requirements
Objective

This test case is intended to verify the following implied or specified requirements:

  • rtnv shall be able to display the main menu within 20 seconds of selecting the app from the Android menu given 20 stocks in the portfolio and access to the internet.
  • rtnv shall be able to store at least 20 stocks in the portfolio.
  • When the calculate net value button is pressed, the sum of the products of the number of shares and value of each share shall be calcuated and that total displayed (in dollar format) in the calculated net value display field.
  • rtnv shall use a public server such as Yahoo finance to retrieve the closing prices of stocks identified by the ticker symbols. Note: this is a demonstration app only and is not meant to be massively distributed for large scale use.

The objective of this test case is to verify that the app can update a portfolio populated to the maximum required size within the time limit of the requirements.
Setup
Actions
  1. Delete the app from the target device.
  2. Compile and load rtnv onto the target Android device.
  3. Verify the compile completes with no errors.
  4. Verify the installation completes with no errors.
  5. Using any method desired, enter 20 valid ticker symbols each with a user defined number of shares.
  6. Select the Android main lauch menu.
  7. Select the Andriod task list and terminate rtnv.
  8. From the launch menu, start rtnv and note the time.
  9. Press the Calculate Net Value button and determine when the net value stops changing.
  10. Verify that time is less than 20 seconds.
  11. Press the "Next" button and walk through the portfolio verifying that every entry has a non zero share value.
  12. Record the product of the number of shares and value of each share. Sum those products and verify that they equal the value displayed in the "Calculated Net Value" field.
Expected The twenty stocks will be be updated within 20 seconds and the calcuated net value will match the value calculated by hand.
Results FAILED
Cleanup
  • Unnecessary
Requirements RTNV-PERF-010, RTNV-PROV-020, RTNV-PR-CNV-010, RTNV-EXT-IF-060
