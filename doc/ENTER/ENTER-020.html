Test Case ENTER-020
Title Pressing Enter when there is an Invalid Ticker Symbol
Objective

This test case is intended to verify the following implied or specified requirements:

  • When the enter button is pressed, rtnv shall send a retrieve request to the public server if the ticker field is not empty.
  • rtnv shall store a persistent copy of the ticker, number of shares, and value of each share in the persistent database.
  • rtnv shall store zero as the value of a share if the value can not be retrieved by the public server.
  • rtnv shall clear the ticker, number of shares, and value of each share fields at completion of processing.

The objective of this test case is to verify the requirements identified above which cover the entry of a stock into the portfolio which does not match an entry that can be retrieved by the public server.
Setup
  • Test case ENTER-010 was immediately executed prior to this test case.
Actions
  1. Select the "Ticker Symbol" field and enter the ticker value of AAAB.
  2. Select the "Number of Shares" field and enter the value 1.
  3. Press the "Enter" button.
  4. Verify a "blank" screen is displayed.
  5. Press the "Next" button and verify that the symbol AAPL is displayed with 200 shares and that a non zero value is displayed for the last closing price of the stock.
  6. Press the "Next" button again and verify that the symbol INTC is displayed with 500 shares and that a non zero value is displayed for the last closing price of the stock.
  7. Press the "Next" button again and verify that the symbol AAAB is displayed with 1 share and that "Not found" is displayed for the last closing price of the stock.
  8. Press the "Next" button again and verify that a "blank" screen is displayed.
Expected With entry of a ticker that doesn't match anything accessible by the public server, the closing value will be displayed as "Not found".
Results PASSED
Cleanup
  • Unnecessary
Requirements RTNV-PR-ENTER-010, RTNV-PR-ENTER-010.001, RTNV-PR-ENTER-010.004, RTNV-PR-ENTER-010.005
