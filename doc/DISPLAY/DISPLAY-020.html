Test Case DISPLAY-020
Title Number of shares field
Objective

This test case is intended to verify the following implied or specified requirements:

  • rtnv shall have a displayable and editable field that allows a user to enter the number of shares owned of a given ticker symbol.

The objective of this test case is to verify that the requirements specified above related to the attributes of the stock number of shares field are implemented and funtioning.
Setup
Actions
  1. Using the display initiated by test case DISPLAY-010, select the "Number of shares field" (Number of Shares).
  2. Verify a numeric keypad is displayed and that numbers (0-9) and the decimal point (.) can be entered.
Expected The field that contains the number of shares should allow the user to enter whole and factional values that store the number of shares held in the portfolio.
Results PASSED
Cleanup
  • Unnecessary
