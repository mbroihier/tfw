Test Case DISPLAY-010
Title Stock ticker field and Buttons
Objective

This test case is intended to verify the following implied or specified requirements:

  • rtnv shall have a displayable and editable field that allows a user to enter stock ticker symbols.
  • rtnv shall have buttons labeled “enter”, “delete”, “next”, and “calculate net value” that conform to the processing requirements identified below.

The objective of this test case is to verify that the requirements specified above related to the attributes of the stock ticker field are implemented and funtioning.
Setup
Actions
  1. Delete the app from the target device.
  2. Compile and load rtnv onto the target Android device.
  3. Verify the compile completes with no errors.
  4. Verify the installation completes with no errors.
  5. Verify that a screen is displayed that has a text field for the ticker symbol, a text field for the number of shares, and a text field for the value of each share.
  6. Verify that there are four buttons: Enter, Delete, Next, and Calculate Net Value.
  7. Verify that no runtime errors are splashed to the screen.
  8. Select the "Stock ticker" (Ticker Symbol) field.
  9. Verify that a keyboard is displayed and that characters (A-Z) and numbers (0-9) can be entered to construct ticker symbols.
Expected The field that contains the ticker symbol should be editable with the ability to enter alpha numeric characters into the field. There should be "enter", "delete", "next", and "calculate net value" buttons.
Results PASSED
Cleanup
  • Unnecessary
Requirements RTNV-EXT-IF-010, RTNV-EXT-IF-050
